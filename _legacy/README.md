# Legacy static landing (archived)

The previous **single-file** site lives in `legacy-landing.html`. The production marketing site is now the **Next.js** app in the repo root (`npm run dev`).

To preview the old file only:

```bash
cd _legacy && python3 -m http.server 4321
```

Open [http://localhost:4321/legacy-landing.html](http://localhost:4321/legacy-landing.html).
